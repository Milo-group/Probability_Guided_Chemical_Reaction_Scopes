This folder contains five files and a subfolder detailing the models presented in the Supporting Information (SI).

**Files Overview**  
  
```Best.Model.Analysis.Deuteration.pdf```  
Contains an analysis of the selected model from the paper, focusing on feature importance.  

```Model.Search.Aldehyde.Deuteration.Full.pdf```  
Presents a model search using ordinal logistic regression on the full dataset (no train/test split).  

```Model.Search.Aldehyde.Deuteration.pdf```  
Presents a model search using ordinal logistic regression after applying similarity-based under-sampling.  

```Model.Search.Full.Non.OrdinalAldehyde.Deuteration.pdf```  
Presents a model search using non-ordinal logistic regression on the full dataset (no train/test split).  

```Model.Search.Non.OrdinalAldehyde.Deuteration.pdf```  
Presents a model search using non-ordinal logistic regression after applying similarity-based under-sampling.  

**Subfolders Overview**  
  
**Variable impact**  
Analysis of the best model, including feature removal analysis with explanations.
