In this folder there are 3 ```*.csv``` files and 1 ```*.R``` file:

**External_Validation_Data.csv**
Contains the external validation dataset used to test model performance on unseen data.

**Predicting_New_Substrates_Data.csv**
Contains new substrate data for prediction using the trained models.

**Training_Data.csv**
Contains the training dataset used to build the models.

**ConvexHall.R**
R script for generating convex hull analyses of the datasets.
