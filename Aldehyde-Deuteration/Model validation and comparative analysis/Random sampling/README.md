This folder contains 3 ```*.csv``` files, 1 ```*.Rmd``` file, and 1 Word document:

**External_Validation_Data.csv**  
Contains the external validation dataset used to test model performance on unseen data.

**Predicting_New_Substrates_Data.csv**  
Contains new substrate data for prediction using the trained models.

**Training_Data.csv**  
Contains the training dataset used to build the models.

**RandomSampling_Final.Rmd**  
R Markdown file containing the code and workflow for the random sampling analysis.

**RandomSampling_Final.docx**  
Word document presenting the results of the random sampling analysis.
