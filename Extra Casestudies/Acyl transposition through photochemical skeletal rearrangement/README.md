In this folder there is one subfolder and one file:  
  
**Data**  
Contains the ```computed_descriptors_Sarpong2025.csv``` dataset used for modeling. It includes molecule tags, SMILES for each molecule, all features, and the class output.  
  
```ModelConstruction_Sarpong2025.R```  
R script that allows the recreation of the results reported in the SI.

```Cross validation_Sarpong2025.R```  
R script that allows the recreation of the cross-validation analysis as reported in the SI.
