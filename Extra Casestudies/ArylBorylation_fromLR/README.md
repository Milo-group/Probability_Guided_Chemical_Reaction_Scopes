In this folder there is one subfolder and two files:  

**Data**
Contains the ```Stevens_data_organized_for_classification.csv``` dataset used for modeling. It includes molecule tags, InChIs for each molecule, all features, and the class output.

```ModelConstruction_ArylBorylation.R```  
R script that allows the recreation of the results reported in the SI.
  
```Cross validation_ArylBorylation.R```  
R script that allows the recreation of the cross-validation analysis as reported in the SI.
