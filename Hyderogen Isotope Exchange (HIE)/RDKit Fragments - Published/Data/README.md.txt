In this folder there are 3 ```*.csv``` and 2 ```*.R``` files:  
  
**Predictions_Data.csv**  
Data corresponding to all predictions tested by the model.  
    
**Sampling_Data.csv**  
Data for the similarity-based under-sampling.  
  
**Training_Data.csv**  
Training data for the model.  
  
**Cross validation_HIE.R**  
Cross-validation code and analysis for the HIE case.  
  
**Model construction_HIE.R**  
Model search code, including confusion matrices and probability heatmaps for the HIE case.
