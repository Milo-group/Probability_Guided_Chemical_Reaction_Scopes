This folder contains two files and three subfolders detailing the models presented in the Supporting Information (SI) and menuscript.  
  
**Files Overview**  
  
```Best.Model.Feature.Analysis.RDKit.Fragments```  
Contains an analysis of the selected model from the paper, focusing on feature importance.  
  
```CH.Activation.Top10.Models.RDKit.Fragments```  
Presents the model search using non-ordinal logistic regression after applying similarity-based under-sampling, as published in the manuscript.  
  
**Subfolders Overview**  
  
**Variable impact**  
Analysis of the best model, including feature removal analysis with explanations.  
    
**Binary_SI**  
Model containing three features for a binary classification option that can be found in the SI.  
    
**3Class_SI**  
Model containing three features for a multinomial classification option (without the null class) that can be found in the SI.
